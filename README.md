# 🚀 Selenium Automation Framework (POM)

## 📖 About Project
This is a Selenium automation testing framework built using the Page Object Model (POM) design pattern.  
It is designed to automate web application testing with proper structure, maintainability, and reusable components.

---

## 🛠️ Tech Stack
- Selenium WebDriver  
- Python / Java  
- PyTest / TestNG  
- Page Object Model (POM)  
- Git & GitHub  
- HTML Reports  

---

## ⚙️ Features
- POM-based framework structure  
- Automated test execution  
- HTML test report generation  
- Reusable and maintainable code  
- Error handling and debugging  
- Git version control integration  

---

## ▶️ How to Run
1. Clone the repository  
   git clone <repo-link>

2. Install dependencies  
   pip install -r requirements.txt

3. Run tests  
   pytest

---

## 📊 Reports
HTML reports are generated after execution inside the reports folder.  
Open them in browser to view test results.
